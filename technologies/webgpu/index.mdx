# WebGPU

WebGPU is a low-level, high-performance graphics and compute API for the web. It offers a modern, efficient, and cross-platform solution for accessing graphics and compute capabilities of modern GPUs from web browsers. Its compute capabilities have significant implications for AI applications on the web.
